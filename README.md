# Bare-Shell  
A minimal POSIX-like shell that executes programs and supports `exit`. No scripting, pipes, or redirections.  

**Build & Run:** 
```
gcc bare-shell.c -o bare-shell && ./bare-shell
```  
**Usage:**  
- `exit` → Quits shell
- `exec filename` → for run exec file
